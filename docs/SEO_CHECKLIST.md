# SEO Checklist (Sanpablo)

## Technical
- [ ] robots.txt correct for dev (block indexing if dev)
- [ ] WordPress setting: discourage search engines (secondary)
- [ ] Canonical tags: correct host
- [ ] XML sitemap: enabled + reachable
- [ ] No mixed-content + HTTPS enforced

## On-page
- [ ] Home title + meta description match brand promise
- [ ] One H1 per page; clean hierarchy
- [ ] Product category pages have unique copy
- [ ] Image alt text for key banners/products

## Structured data
- [ ] Product schema on PDP
- [ ] Organization schema
- [ ] Breadcrumb schema

## Measurement
- [ ] Add/verify Google Search Console + submit sitemap (prod)
- [ ] Add/verify GA4 (if desired)
